IOS Application Requirements:

1.Pod Installations: 

 pod 'Firebase/Core'
    
 pod 'Firebase/Database'

 pod 'Firebase/Auth'
    
 pod 'Firebase/Storage'

2. Microsoft Vision API Keys to access the API. Include those in the Server.py code provided in the documentation. 

3. Firebase Account to store OBDII Reader Images  

4. 'Server.py'  provided should be running when the app is in use and the address of the server needs to be mentioned in the 'viewController.swift' file before launching the application.
